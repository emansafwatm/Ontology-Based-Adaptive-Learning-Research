# Historical Adaptive Examination System

This directory preserves a sanitized copy of the historical ASP.NET/C# implementation. Database files and credentials are intentionally excluded.

The original code used SQL Server, ASP.NET membership services, and OwlDotNetApi. It may require an older .NET Framework environment and unavailable database schemas.

Security-sensitive strings and local ontology paths were replaced by placeholders. SQL statements and other patterns remain representative of historical code and should not be reused in production without modernization and security review.

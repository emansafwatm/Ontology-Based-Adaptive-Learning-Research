# Excluded Materials

The source archive contained materials unsuitable for public GitHub distribution. These were excluded:

- SQL Server and Access database files
- ASP.NET membership databases and log files
- Original `web.config` files and hard-coded credentials
- Compiled binaries and debug symbols
- NuGet/package caches and third-party library copies
- Visual Studio user/cache files
- Duplicate ZIP/RAR archives and backups
- Temporary Microsoft Office files
- Presentations named for students or potentially containing identifiable records
- Downloaded presentation templates and unrelated external assets

The omission of a file does not imply that it was malicious or invalid; exclusions were conservative to reduce privacy, licensing, security, and repository-quality risks.
